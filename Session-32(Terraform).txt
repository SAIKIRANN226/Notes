Data-sources
--------------
Where the data-sources will be useful?To query data from the provider and also from the existing resource,but to query the existing resource we need to give some input like vpc_id,this vpc_id will not come from data-sources,only default vpc_id we can take.but to take your created vpc_id is again data-sources is not an option,so we have ssm parameter is like configuration storage for example one application is there in this application we have configuration and that can be stored in the ssm parameter store,and another application can refer that configuration from the ssm parameter,and also stores.and another application like that so on.It is a central store,So that means you need to store the vpc_id configuration in the ssm parameter by writing a small script(parameters.tf in roboshop-terraform).
